# Task 1 - Basic Web Development Project

## Project Title

Personal Introduction Website

## Description

This project is developed as part of the ApexPlanet Web Development Internship Task-1. The website demonstrates the basic concepts of HTML, CSS, and JavaScript.

The website includes:

* A welcome heading
* An introduction paragraph
* An image
* A hyperlink
* CSS styling for better appearance
* A JavaScript button that displays an alert message

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

* Responsive and clean design
* Attractive styling using CSS
* Interactive button using JavaScript
* Easy navigation with hyperlinks

4. The website will open in your browser.
